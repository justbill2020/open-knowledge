import { Clock, Undo2 } from 'lucide-react';

The timeline shows you the history of edits made to the document you have open. From there you can view or revert to older versions of the document.

## What the timeline shows

Open the timeline from the document side panel by clicking the <Clock role="img" aria-label="Timeline" className="inline size-4 align-text-bottom" /> icon. Each entry shows who made the change, when it happened, and a short summary of the change.

Timeline entries include:

- **Agent and human edits** with contributor attribution and change summaries
- **Upstream sync** entries from Git
- **File system** changes made outside of Open Knowledge

## Review changes

Click a timeline entry to expand it. Open Knowledge shows an inline diff so you can compare that version with the current document. If that version is the same as the current document, you will see `no changes`.

You can switch between unified and split diff layouts from the editor controls.

![An expanded timeline entry showing a split diff next to the document, with removed lines highlighted in red and added lines in green](/screenshots/timeline-diff-view.png)

## Restore a previous version

Use the restore button <Undo2 role="img" aria-label="Restore" className="inline size-4 align-text-bottom" /> on a timeline entry to restore the document to the selected version. Restoring is append-only: Open Knowledge creates a new change with the old content instead of deleting history.
